# components

React 组件目录，包含所有视图和 UI 组件。

核心职责：
- 承载应用的主要视图组件
- 组织 shadcn/ui 组件库
- 提供可复用的业务组件

依赖关系：主应用 → 视图组件 → UI 基础组件

> 一旦本文件夹有所变化，请更新本文档。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `app-sidebar.tsx` | 核心 | 应用侧边栏导航，包含菜单和用户信息 |
| `canvas-view.tsx` | 核心 | 画布编辑器视图，承载 ReactFlow 和 AI 聊天 |
| `project-list.tsx` | 核心 | 项目列表视图，网格展示所有项目 |
| `nav-user.tsx` | 辅助 | 侧边栏用户头像和下拉菜单 |
| `ui/` | 辅助 | shadcn/ui 基础组件库 |
