# components/ui

shadcn/ui 基础组件库，基于 Radix UI 和 Tailwind CSS。

核心职责：
- 提供可复用的 UI 基础组件（按钮、输入框、对话框等）
- 统一的设计系统和样式规范
- 无障碍访问支持

依赖关系：业务组件 → UI 基础组件 → Radix UI 原语

> 一旦本文件夹有所变化，请更新本文档。
